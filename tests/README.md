# 🛍️ ShopEasy

A sample React e-commerce app with full Playwright test automation.

![Playwright Tests](https://github.com/Sindhu-AI-1/shopeasy/actions/workflows/playwright.yml/badge.svg)

## Tech Stack
- React + Vite
- Playwright + TypeScript
- Page Object Model
- GitHub Actions CI/CD

## Test Coverage
- 53 UI tests across Login, Products, Cart
- 11 API tests
- Cross browser: Chrome, Firefox, Safari
- Data-driven tests with for loops
